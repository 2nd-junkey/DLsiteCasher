# tl;dr

DLsiteの物量に沈んで全部nasにぶん投げた人へ贈る

# install

`./Tools/install.sh //nas/DLsite ./Storage`

# usage

- Tools/install.sh
  
  - `Tools/convert.sh`を実行後、指定したディレクトリにsubmoduleのリンクを生成
  
- Tools/convert.sh
  
  - nasに上げたデータを全部ベアリポジトリに交換

- Tools/uncommitlist.sh
  
  - 未コミット一覧を表示

- Tools/pushAll.sh
  
  - 未プッシュ一覧を表示、確認後push

- Tools/clear.sh
  
  - pushしてから一定期間たったものを削除
